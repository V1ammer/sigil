# messenger-server

Корпоративный E2E-мессенджер — серверная часть.

Сервер выступает в роли Delivery Service для MLS (RFC 9420): хранит зашифрованные
сообщения, управляет аутентификацией по ключам и роутингом сообщений между устройствами
участников групп. Не имеет доступа к содержимому сообщений.

## Разработка

### Dev shell

```sh
# Через direnv (рекомендуется):
direnv allow

# Или вручную:
nix develop
```

### Запуск сервера

```sh
cargo run -p messenger-server
```

По умолчанию слушает `http://127.0.0.1:8080`.

Проверка:

```sh
curl http://127.0.0.1:8080/health
# → ok
```
