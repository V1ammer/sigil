# APK Changelog

## 2026-06-16 — Усиление E2E

- Сообщения никогда не отправляются открыто: при невозможности зашифровать —
  отмена отправки, а не plaintext.
- Мультиустройство: чат читается со всех устройств.
- Post-compromise security: периодический rekey ключей.

## 2026-06-16 — Группы

- Групповые чаты на настоящем MLS: создание, управление участниками, имя, аватар.
- Личные чаты переведены на настоящий E2E.

## 2026-06-16 — Уведомления

- Уведомления о входящих (системные + веб), с учётом настроек. Без FCM.
