# Changelog

Короткие заметки по релизным сборкам APK (новое сверху).

## 2026-06-16 — Группы

- Групповые чаты на настоящем MLS: создание с N участниками, имя и аватар (E2E).
- Управление участниками: добавить / убрать / выйти; только владелец управляет
  составом, при выходе владелец передаёт права выбранному участнику.
- Личные чаты переведены на настоящий E2E (раньше конверт сообщения уходил
  открытым); старые личные чаты не трогаются.
- Системные плашки «вошёл / вышел / удалён» в группах.
- Аватар и имя группы видны в списке чатов и шапке.

## 2026-06-16 — Уведомления

- Реальные уведомления о входящих: системные (Android) + веб Notification API.
- Срабатывают, когда чат не открыт или приложение свёрнуто; уважают настройки
  (звук, превью, «не беспокоить», mute чата). Без FCM — только пока приложение живо.
